**Exercício OpenCV:**

Crie um programa em Python utilizando a biblioteca **OpenCV** que:

1.  carregue uma imagem do computador;

2.  converta a imagem para **escala de cinza**;

3.  exiba a imagem na tela;

4.  calcule o **histograma da imagem**;

5.  mostre o histograma em uma janela gráfica.

O histograma deve representar a quantidade de pixels para cada nível de intensidade, variando de **0 a 255**.
